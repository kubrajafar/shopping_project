# Shopping Project

Welcome to the Shopping Project! This web application allows users to browse and purchase various products online.

## Live Demo

You can access the live demo of the project [here](https://kubrajafar.github.io/shopping_project/).

## Features

- **User Authentication**: Users can create accounts, log in, and log out.
- **Product Browsing**: Users can browse through different product categories and view detailed information about each product.
- **Shopping Cart**: Users can add products to their shopping cart and proceed to checkout.
- **Payment Processing**: Secure payment processing functionality is integrated for seamless transactions.
- **Order History**: Users can view their order history and track the status of their orders.
- **Responsive Design**: The application is fully responsive, ensuring a seamless experience across different devices.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Firebase (for backend functionality)

## Usage

1. Navigate to the live demo [here](https://kubrajafar.github.io/shopping_project/).
2. Sign up for a new account or log in using the provided credentials.
3. Browse through the available products, add items to your cart, and proceed to checkout.
4. Complete the payment process and track your order status in the order history.
